# Encryption Tool

```markdown
Welcome to the Encryption Tool, your simple solution for secure text and file processing, powered by **HTML, JavaScript**, and **C**.

---

## Features
- **Encrypt/Decrypt Text**: Securely transform text to protect sensitive information.
- **Encrypt/Decrypt Files**: Command-line functionality for robust file security.

---

## Technologies Used
- **HTML & JavaScript**: Build the interactive web interface.
- **C Language**: Enable fast and secure file encryption.

---

## How to Use

### Web Version
Visit our [Encryption Tool Website](https://sites.google.com/diu.edu.bd/encryption-tool/home) and follow these steps:
1. Choose your action (**Encrypt** or **Decrypt**).
2. Adjust the "Condition Number" to set encryption strength.
3. Enter text and click **Process** for results in seconds.

### Daily Applications
- **Secure Personal Data**: Safeguard sensitive information easily.
- **Password Management**: Encrypt your passwords for added safety.
- **Safe Sharing**: Protect shared files with encryption before transfer.
```
